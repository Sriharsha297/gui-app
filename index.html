IMAGE PROCESSING EDGE DETECTION TECHNIQUE FOR TRAFFIC CONTROL PROBLEM



  • The frequent traffic jams at major junctions call for an efficient traffic management system in place.
  • The wastage of time and increase in pollution levels can be eliminated on a city-wide scale by this system. Image processing is a better technique to control the state change of the traffic light. It shows that it can reduce the traffic congestion and avoids the time being wasted by a green light on an empty road. It is also more efficient because it uses actual traffic images.
  • This project proposes to implement a smart traffic controller using real time image processing. The image sequences from a camera are analyzed using edge detection Algorithms.
